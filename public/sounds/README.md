# Sons de notification

## notification.mp3

Ce fichier doit être un fichier audio MP3 de notification.

Vous pouvez :
1. Créer votre propre son de notification
2. Utiliser un son libre de droits depuis des sites comme:
   - https://freesound.org/
   - https://notificationsounds.com/
   - https://mixkit.co/free-sound-effects/notification/

Le son doit être court (1-2 secondes) et agréable.

**Format requis:** MP3  
**Nom du fichier:** notification.mp3  
**Emplacement:** /public/sounds/notification.mp3
